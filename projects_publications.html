Personal Projects

C++ Personnel Directory

Github repository: Personnel Directory Project

Status: In Progess:
Small scale Object Orientated Project in C++. A Peronnel Directory with nodes that holds a person's information that is put into a linked list. A specific person's node can be access on a doubly linked list using either their name or ID number within the system. Users may also view any relationships with this person (whether coworker or family) and display a certain relationship's information. However, while on a person's node, no other person's node may be modified unless the user switches over to that person's node.

Object Orientated Detail Animation Application

Github repository: OOD Animation Project

Status Complete:
Small project where a team of 2 developed an animation app with MVC (Model, View, Controller) in mind, where clients could create an animation project with basic shapes that could rotate, translate, or resize. Clients could ask for the model in plain text, SVG, or JFrame. As client made changes through the view, directions passed through the controller and changed the model.

Radiosity Lighting Utilizing OpenGL

Github repository: Radiosity Lighting Project

Status: Complete
Mid scale project created by team of 2 using c++ and OpenGL. The idea was to show how the lighting on objects changed as they translated, rotated, and resized across a room with different fixed "light points". Was able to utilize a lot of code created throughout semester to parse object files and mtl files for proper texturing.
